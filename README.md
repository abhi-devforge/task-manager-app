# Task Manager App
This is a small web project I built while learning the fundamentals of HTML, CSS, and JavaScript.
The application allows users to add tasks to a list and remove them dynamically.  
The main goal of this project was to understand how JavaScript interacts with the DOM and how basic UI behavior works in a browser.

## What the App Does
- Users can type a task into the input field
- Clicking "Add Task" inserts the item into the list
- Each task includes a delete option
- Tasks update instantly without refreshing the page

## Why I Built This
I created this project as part of my practice to strengthen my core web development skills and become comfortable with GitHub and project structuring.

## Technologies Used
- HTML for structure
- CSS for basic styling
- JavaScript for interactivity

## Notes
This project is intentionally simple and focused on learning concepts rather than visual design or advanced features.
